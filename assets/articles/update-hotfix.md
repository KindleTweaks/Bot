In order to update the hotfix, you can use 2 methods.

Recommended Method: Using `MRPI`

1. Download the latest Hotfix from [here](https://github.com/KindleModding/Hotfix/releases).
2. Copy this `.bin` file to the `mrpackages` folder on your Kindle.
3. Run `;log mrpi` in your search bar.
4. The Hotfix will install, you should see funky text shortly.
5. If no text appears, or `MRPI` whitescreens, try method `2`.
6. Run the `Run Hotfix` or `Run Bridge` booklet once.
7. You're done!

Alternative Method: Using Updates

1. Download the latest Hotfix from [here](https://github.com/KindleModding/Hotfix/releases).
2. Copy this `.bin` file to the Kindle root.
3. Turn on aeroplane mode, delete any `tmp.bin.partials` if there are any.
4. Open KUAL, tap `Rename OTAs`, and click `Restore`.
5. Go to settings, tap the three dots, and press update.
6. The Hotfix will install, wait until rebooted fully.
7. Open KUAL, tap `Rename OTAs`, and click `Rename`.
8. Tap the `Run Hotfix` or `Run Bridge` booklet once.
9. You're done! You can turn off aeroplane mode.

Ensure you're on Hotfix `2.3.2`, `2.3.7` or above!
Others may be unstable.
